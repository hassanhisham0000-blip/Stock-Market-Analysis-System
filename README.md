# StockVision: Stock Market Analysis System 📈

StockVision is a comprehensive tool built to analyze financial markets using real-time data. It provides interactive visualizations and technical indicators to help users understand market trends.

## 🌟 Features
* **Real-time Data:** Fetches live stock information via the Yahoo Finance API.
* **Interactive Visualizations:** Includes Line and Candlestick charts built with Plotly.
* **Technical Analysis:** Calculates Moving Averages (MA7, MA20, MA50) to identify trends.
* **Market Indicators:** Volume analysis and key performance metrics.
* **Modern UI:** A clean, responsive dashboard built using Streamlit.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Plotly, yFinance
* **Framework:** Streamlit
